# AssessPro – платформа для создания и прохождения тестов

AssessPro – это веб-приложение на Java (Spring Boot), которое позволяет создавать тесты, управлять категориями, проходить тесты и получать статистику. В проекте реализованы три роли пользователей: **Администратор**, **Создатель тестов** и **Тестировщик**.

## 🚀 Функциональность

### 👤 Для всех пользователей
- Регистрация и авторизация (Spring Security)
- Заполнение профиля (ФИО)
- Просмотр каталога опубликованных тестов

### 🧑‍🏫 Для создателя тестов (CREATOR)
- Создание, редактирование, удаление тестов
- Добавление вопросов и вариантов ответов
- Публикация / снятие с публикации теста
- Просмотр статистики по своим тестам
- Управление категориями

### 🧪 Для тестировщика (TESTER)
- Прохождение опубликованных тестов
- Просмотр истории своих попыток
- Детальный просмотр результатов
- Поиск тестов по названию и категории

### ⚙️ Для администратора (ADMIN)
- Управление пользователями (создание, блокировка, изменение ролей)
- Просмотр общей статистики приложения
- Экспорт статистики в PDF

## 🛠 Технологический стек

- **Backend**: Java 21, Spring Boot 4, Spring Security, Spring Data JPA, Hibernate
- **База данных**: PostgreSQL
- **Фронтенд**: Thymeleaf, Bootstrap 5, jQuery, HTMX
- **Тестирование**: JUnit 5, Mockito, Testcontainers, AssertJ
- **Документация API**: Swagger/OpenAPI
- **Мониторинг**: Prometheus, Grafana
- **Сборка**: Maven
- **Контейнеризация**: Docker (опционально)

## 📦 Установка и запуск

### Требования
- JDK 21
- PostgreSQL (локально или через Docker)
- Maven
